# 🌍 Global Energy Consumption and Emission Analysis using SQL

## 📌 Project Overview

This project analyzes global energy production, consumption, emissions, GDP, and population data using SQL. The objective is to uncover trends, compare countries, measure sustainability indicators, and generate business insights through advanced SQL queries.

The project demonstrates SQL skills including data retrieval, aggregation, joins, Common Table Expressions (CTEs), subqueries, window functions, ranking functions, and analytical queries.

---

## 🎯 Problem Statement

Global energy demand continues to grow while countries face challenges related to sustainability, economic growth, and carbon emissions.

The objective of this project is to analyze global energy datasets and answer key business questions such as:

- Which countries produce and consume the most energy?
- How do GDP and energy consumption relate?
- Which countries have the highest per-capita emissions?
- How has energy production changed over time?
- What sustainability insights can be derived from the data?

## 📈 Key Insights

- Countries with higher GDP generally consume more energy.
- Per-capita emissions vary significantly across countries.
- Energy production growth is concentrated among major economies.
- Population alone is not a strong predictor of energy consumption.
- Several countries produce significantly more energy than they consume.
- Emission intensity differs considerably across regions.
